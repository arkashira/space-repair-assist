# breakeven.md

## Unit Economics & Break-Even Analysis

### Cost per Active User
- **Compute Costs**: $5/user/month
- **Storage Costs**: $2/user/month
- **Bandwidth Costs**: $1/user/month
- **Total Cost per Active User**: 
  - **Total**: $5 + $2 + $1 = **$8/user/month**

### Pricing Tiers
| Tier          | Price ($/mo) | Features                                                                 |
|---------------|--------------|--------------------------------------------------------------------------|
| Basic         | $15          | Remote guidance, basic simulation tools, access to community support     |
| Professional   | $30          | All Basic features + advanced simulation tools, 3D printing capabilities, priority support |
| Enterprise    | $50          | All Professional features + custom solutions, dedicated account manager, analytics dashboard |

### Customer Acquisition Cost (CAC) Range
- **Estimated CAC**: $50 - $100 per user
  - **Assumptions**: 
    - Marketing spend, sales efforts, and operational costs associated with acquiring a new user.

### Lifetime Value (LTV) Estimate
- **Average Revenue per User (ARPU)**: 
  - Basic: $15/month
  - Professional: $30/month
  - Enterprise: $50/month
- **Average User Lifespan**: 24 months
- **LTV Calculation**:
  - Basic: $15 * 24 = **$360**
  - Professional: $30 * 24 = **$720**
  - Enterprise: $50 * 24 = **$1200**

### Break-Even Users Count
- **Monthly Fixed Costs**: $10,000 (assumed for development, operations, and marketing)
- **Break-Even Calculation**:
  - Break-even users = Monthly Fixed Costs / (Price per User - Cost per User)
  
Using the **Professional Tier** as a reference:
- Break-even users = $10,000 / ($30 - $8) = $10,000 / $22 = **455 users**

### Path to $10K MRR
- **Target Monthly Recurring Revenue (MRR)**: $10,000
- **Using Professional Tier**:
  - Required users = $10,000 / $30 = **334 users**
  
- **Using Basic Tier**:
  - Required users = $10,000 / $15 = **667 users**

- **Using Enterprise Tier**:
  - Required users = $10,000 / $50 = **200 users**

### Summary
- **Total Cost per Active User**: $8/user/month
- **Break-even Users Count**: 455 users (Professional Tier)
- **Path to $10K MRR**: 
  - 334 users (Professional Tier)
  - 667 users (Basic Tier)
  - 200 users (Enterprise Tier)
